## O aplikacji

Odblokowujesz telefon ponad 100 razy dziennie. AutoFiszki wykorzystują każdą z tych
chwil: aplikacja ustawia na ekranie blokady tapetę z fiszkami, więc słówka powtarzasz
przy okazji — bez otwierania czegokolwiek.

- 🔒 **Nauka na ekranie blokady** — fiszki widzisz przy każdym zerknięciu na telefon;
  wygląd tapety (układ, styl, liczbę słów) ustawiasz we wbudowanym edytorze
- 🧠 **Inteligentne powtórki FSRS‑4.5** — nowoczesny algorytm spaced repetition
  pokazuje słowo dokładnie wtedy, gdy masz je zaraz zapomnieć
- 🃏 **10 gotowych talii** — angielski (2859 najczęstszych słów z listy NGSL,
  z transkrypcją IPA), hiszpański (2500 słów z korpusu Tatoeba), stolice świata
  i pierwiastki chemiczne
- 🛡️ **Prywatność bez kompromisów** — aplikacja nie ma uprawnienia dostępu
  do internetu, więc Twoje dane fizycznie nie opuszczają telefonu; działa w pełni offline
- 💎 **Jednorazowy zakup** — zero reklam, zero subskrypcji, wszystkie przyszłe
  aktualizacje w cenie

**Wymagania:** Android 8.0 lub nowszy · aplikacja po polsku

## To repozytorium

Kod strony promocyjnej [autofiszki.pl](https://autofiszki.pl) — statyczny
HTML/CSS/JS bez frameworków i bez procesu budowania, hostowany na GitHub Pages.

Strona, tak jak aplikacja, szanuje prywatność: nie używa cookies, analityki ani
zasobów z zewnętrznych serwerów (fonty hostowane lokalnie), dlatego nie potrzebuje
banera cookie.

### Podgląd lokalny

```
python -m http.server 8000
```

i otwórz http://localhost:8000
